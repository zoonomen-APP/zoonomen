Cunningham's Alaska Ornithologist workers
D. Cunningham's extraordinary documentation of biographical data of Alaskan Ornithological workers.
This is manuscript material. It's use is encouraged with the understanding that all such use it will be acknowledged and that the data has been accessed via zoonomen.net.
Text such as: "Data from Cunningham's Alaska Ornithological workers Biographies accessed via www.zoonomen.net" with a date is appropriate for each instance of use. A generalized blanket acknowledgemnt is held to be unsatisfactory.
