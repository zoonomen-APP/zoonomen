Zoological Type Notes --R

  • Rhynchotus rufescens (Temminck) 1815 Hist.Nat.Pig.Gallin. 3 p.552,747
    • Type by monotypy Rhynchotus fasciatus Spix, 1825 = Tinanums rufescens Temminck, 1815.
    • Type locality: "Brazil and Paraguay, ex Azara; Sao Paulo suggested as the type locality by Hellmayr, 1929, Publ.Field.Mus.Nat.Hist.., Zool. Ser. 12, p.479." Peters 1:36.
  • Ramphastos tucanus Linnaeus 1758 Syst.Nat.ed.10 p.103
    • Type by subsequent designation, Ramphastos erythrorhynchus Gmelin = Ramphastos tucanus Linnaeus (Vigors, Zool.J. 2, 1826, p.471).
    • Type locality (Peters representation): "South America, restricted to Surinam by Griscom and Greenway, antea p.430)." [apparently Griscom and Greenway, Bull.Mus.Comp.Zool., 81, 1937 p.427-430].
    • Type locality: "Habitat in America meridionalis." (Linnaeus).
