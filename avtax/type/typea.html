Zoological Type Notes --A

  • Ardeotis arabs (Linnaeus) 1758 Syst.Nat.ed.10 p.154
    • Type by monotypy.
    • Type locality (Linnaeus) "Habitat in Oriente"
    • Grant CHB & Mackworth-Praed CW. 1935. BBOC 55:89 write:" 
      2. On the Type-locality of Choriotis arabs arabs (Linnaeus).

  All authors quote Linnaeus's "In oriente," and we cannot
find that a more definite type-locality has been fixed for this
Bustard. Linnaeus, Syst. Nat. ed. x. i. 1758, p. 154, gives
one reference only, and that to Edwards, Av. i, 1743, p.12,
pl. 12.
  Edwards states that the live specimen from which he drew
The plate came from Mocha in Arabia Felix.  The correct
type-locality for Choriotis arabs arabs (Linnaeus) is, therefore,
Mocha, Yemen, Sourhter Arabia."
2003.02.15
  • Apteryx australis Shaw 1813 Nat.Misc. 24 pl.1057
    • Type by monotypy.
    • Type locality (Peters [Mayr] representation): "New Zealand. Type from Dusky Sound, South Island. fide Oliver, 1955, N.Z. Birds, ed. 2 p.51." Peters 1:10.
  • Aepypodius bruijnii (Oustalet) 1880 Compt.Rend. 90 p.906
    • Type by subsequent designation. Talegallus bruijnii Oustalet (Salvadori, Orn.Papu.Moluc. 3 1882 p.251).
    • Type locality (Peters representation): "Waigeu, locality probably erroneous." Peters 2:9.
  • Ammoperdix heyi (Temminck) 1825 Pl.Col. livr.55 pl.328,329
    • Type by subsequent designation. Perdix heyi Temminck (Reichenbach, Av.Syst.Nat.(1852) 1853 p.xxviii).
    • Type locality (Peters representation): "Desert of Akaba, Arabia." Peters 2:59.
  • Alectoris barbara (Bonnaterre) 1791 Tabl.Encyc.Meth.Orn. 1 p.208 pl.94 fig.2
    • Type by monotypy. Perdix petrosa auct. not of Gmelin = Perdix barbara Bonnaterre.
    • Type locality (Peters representation): "No locality given = Morocco, ex Edwards pl.70." Peters 2:67.
  • Arborophila torqueola (Valenciennes) 1825 Dict.Sci.Nat. 38 p.435
    • Type by monotypy. Perdix olivacea Latham = Perdix torqueola Valenciennes.
    • Type locality (Peters representation): "Bengal" Peters 2:98.
  • Agelastes meleagrides Bonaparte 1850 PZS(1849) p.145
    • Type by monotypy. Agelastes meleagrides Bonaparte.
    • Type locality: "No locality given = Gold Coast". Peters 2:134.
  • Anser anser (Linnaeus) 1758 Syst.Nat.ed.10 p.123
    • Type by tautonomy. Anser domesticus Brisson = Anas anser Linnaeus.
    • Type locality: "Europe and northern North America; restricted to Sweden by Linnaeus, 1761, Fauna Svecica, ed.2 p.40" Peters 1:438.
  • Aix sponsa (Linnaeus) 1758 Syst.Nat.ed.10 p.128
    • Type by subsequent designation. Anas sponsa Linnaeus.
    • Type locality: "in America spetontrionali = South Carolina" AOU CL ed.7 p.65.
  • Anas platyrhynchos Linnaeus 1758 Syst.Nat.ed.10 p.125
    • Type by subsequent designation. (Lesson, 1828). Anas boschas Linnaeus = Anas platyrhynchos Linnaeus.
    • Type locality: "in Europe maritimis = Sweden" AOU CL ed.7 p.68.
  • Aythya marila (Linnaeus) 1761 FaunaSvecica.ed.2 p.39
    • Type by monotypy. Anas marila Linnaeus.
    • Type locality: "in Lapponica = Lapland" AOU CL ed.7 p.76.
  • Aplonis fusca† Gould 1836 PZS Pt4 no.43 p.73
    • Type by subsequent designation. Aplonis fusca Gould (Gray,GR 1840 ListGen.Birds p.30).
    • Type locality (Peters representation): "Murrumbidgee River, Australia [error for Norfolk Island]. Peters 15:79.
  • Actinodura egertoni Gould 1836 PZS Pt4 no.38 p.18
    • Type by monotypy and original designation.
    • Type locality: Nepal. Peters 10:391.
  • Aprosmictus erythropterus (Gmelin) 1788 Syst.Nat. 1 pt1 p.343
    • Type by subsequent designation. Psittacus erythropterus Gmelin (Gray,GR 1846 Gen.Birds 2 p.408). Peters 3:247
    • Type locality: "No locality = New South Wales, apud Matthews." Peters 3:248.
  • Andigena hypoglauca (Gould) 1833 PZS Pt1 no.6 p.70
    • Type by subsequent designation. Pteroglossus hypoglaucus Gould ( Gray,GR 1855 Cat.Gen.Subgen.Birds p.85) Peters 6:80
    • Type locality: "No locality = Colombia designated by Brabourne and Chubb " Peters 6:81.
  • Aulacorhynchus sulcatus (Swainson) 1820 Q.J.Sci.Lit.Arts 9 p.267
    • Type by subsequent designation. A. sulcatus (Swainson) = Pteroglossus sulcatus Swainson (G.R. Gray ListGen.Birds 1840 p.50)
    • Type locality (Peters representation): "Venezuela." Peters 6:70.
