Zoological Type Notes --F

  • Francolinus francolinus (Linnaeus) 1766 Syst.Nat.ed.12 p.275
    • Type by tautonomy. "Francolinus vulgaris Stephens = Tetrao francolinus" Linnaeus.
    • Type locality: "Italia, Oriente, Africa, Asia". (Linnaeus).
    • Type locality (Peters representation): '"Italia, Oriente, Africa, Asia." Restricted type locality, Cyprus ex Edwardsian reference' Peters 2:68.
  • Falcipennis falcipennis (Hartlaub) 1855 J.Orn. 3 p.39
    • Type by monotypy. "Falcipennis hartlaubi Elliot = Tetrao falcipennis Hartlaub."
    • Type locality: Southern shores of the Sea of Okhotsk and the Stanovoi Mountains. Peters 2:37.
