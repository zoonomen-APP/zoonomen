Zoological Type Notes --O

Spizella arborea ochracae Type
  • Described as Spizella monticola ochracea.
  • Collected Nov.8, 1881 by Captain Bendire at Fort Walla Walla, Washington Territory while he was stationed there in the autumn and winter 1881-1882.
  • [2010.06.12] The original description refers to the type specimen, and other birds from the west (presumably in Bendire's collection or other collections?).
  • The original citation can be seen here:

  • Ortalis motmot (Linnaeus) 1766 Syst.Nat.ed.12 p.271
    • Type by original designation. Phasianus motmot Linnaeus.
    • Type locality: "Brasilia, Guiania" (Linnaeus)
    • Type locality (Peters representation): "Brasil and Guiana = Cayenne, accepted as type locality, by Berlepsch and Hartert, 1902, Nov.Zool. 9 p.120." Peters 2:16.
  • Odontophorus gujanensis (Gmelin) 1789 Syst.Nat. 1 pt2 p.767
    • Type by monotypy "Tocro de Guyane" Buffon = Tetrao gujanensis Gmelin.
    • Type locality: "Cayenne and Guiana; Cayenne accepted as type locality by Berlepsch and Hartert, Nov. Zool. 9, 1902, p.121." Peters 2:52
  • Oxyura jamaicensis (Gmelin) 1789 Syst.Nat. 1 pt2 p.519
    • Type by monotypy Anas rubidus Wilson = Anas jamaicensis Gmelin.
    • Type locality: "in Jamaica".
