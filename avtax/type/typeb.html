Zoological Type Notes --B

Burhinus Type
  • Listed by Peters Checklist 2:293 as 
    Type, by monotypy, Charadrius magnirostris Latham
  • The listing for Burhinus magnirostris (Latham) indicates 
    Replaces Charadrius grallarius Latham.
    without presenting a rationale or explanation.
  • HBW 3:350 (Hume) discusses this, indicating that Latham described two taxa magnirostris and grallarius, both of which turned out to be the same species. He goes on to indicate his interpretation that Gould's adoption in 1845 of grallarius (citation not given) constitutes an action as a first reviser, thus fixing grallarius as the name of the type.

Buccanodon Type
  • The citation for this genus is often given as

    Buccanodon (m.) Hartlaub 1857 Syst.Orn.Westafr. p.171

    (e.g. Peters Checklist 6:44, and CBBM 19:38).
  • In Hartlaub's use of the genus the type species was Barbatula DuChaillui Cassin, 1855.
  • However, the name was published by Gray in 1855, in an addendum to the Cat.Gen.subgeneraBirds, and there he used the type as Buccanodon formosus Verr. which I interpret to be a synonym of Barbatula DuChaillui Cassin.
  • I expect that both Gray and Hartlaub took the name Buccanodon as a manuscript name from Verreuax.
  • An argument can be made that Buccanodon Gray, is a nomen oblitum, however of interest in Neave's compilation of Generic names he lists only the Gray 1855 name (p.495, 1939). Obviously when the name is used in combination in a binomen it is difficult, or indeed impossible to determine if the Gray name of the Hartlaub name is in the mind of the author.
  • Neave's listing of the name does not, I suppose constitute "use" (which would eliminate the possibility of it's being a nomen oblitum, but any one who used the name Buccanodon and, for example looked it up in Neave to check the spelling, would, in fact be using Buccanodon Gray, not Buccanodon Hartlaub.
  • However, Sclater 1930 Syst.Av.Aethiopicarum p.859 cites the genus to Gray 1855 CAt.Gen.Subgen.Bds. Addenda. p.1, and lists the type as "Buccanodon (i.e. Barbatula) formosus Verr.= Barbatula duchaillui Cass.
  • The Richmond Index indicates that the Verreaux "entity" is actually
    • Barbatula formosa Jules & E. Verreaux. Rev. Mag. Zool.(2) 7:218, pl.5 bis. May 1855.
    This would appear to possibly have priority over Barbatula Duchaillui Cassin which was presented at the Meeting of Apr. 24, 1855; the publication of this was listed by the PANSP in their 1913 index as occuring on June 15, 1855.
  • Many thanks to Storrs Olson for bringing this material to may attention.

  • Balearica pavonina (Linnaeus) 1758 Syst.Nat.ed.10 p.141
    • Type by monotypy, Balearica Brisson = Ardea pavonina Linnaeus.
    • Peters Checklist 2:154 does not give a type locality.
    • Grant CHB, Mackworth-Praed CW. 1941. BBOC 62:13 discuss the problem of the type location. Their note reads:
        "Linnaeus, Syst. Nat. i. 1758, p.141, gives only Africa, but also
gives references to Aldrovandi, Orn. vol. iii. 1599, lxx. chap. 6;
Willoughby, Orn. 1676, p.201, pl.48; and two others.
Aldrovandi gives no locality, but Willoughby states that
this bird comes from Cape Verde.
  We can, therefore, fix the type-locality of Balearica pavonina
pavonina (Linn.) as Cape Verde, Senegal."

      • Type by subsequent designation Anas bernicla Linnaeus. (Bannister 1870).
      • Type locality: "in Europa boreali = Sweden" AOU CL 7ed. p.60.
    • Bucephala albeola (Linnaeus) 1758 Syst.Nat.ed.10 p.124
      • Type by original designation Anas albeola Linnaeus.
      • Type locality: "in America = Newfoundland" AOU CL 7ed. p.82.
