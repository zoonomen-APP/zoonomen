Zoological Type Notes --S

  • Struthio camelus Linnaeus 1758 Syst.Nat.ed.10 p.155
    • Type by subsequent designation. G.R. Gray, 1840, List Gen. Birds p.63.
    • Type locality: "Syria, Arabia, Libya, Africa" (Linnaeus).
    • Type locality (Peters [Mayr] comments): "restricted to North Africa by Rothschild, 1919, BBOC 39 p.83; further restricted to Sennar by Stresemann, 1926, Orn.Monatsb. 34 p.139." Peters 1(2):4.
  • Saucerottia saucerrottei (Delattre & Bourcier) 1846 Rev.Zool. 9 p.311
    • Type by original designation. Saucerottia typica Bonaparte = Trochilus saucerrottei Delattre and Bourcier.
    • Type locality: Cali, Colombia
  • Scleroptila levaillantii (Valenciennes) 1825 Dict.Sci.Nat. 38 p.441 ? 1826 v.i
    • (Indicated as Type by Cat.BirdsBr.Museum 22 p.127.
    • Type locality (Peters representation): "Cape of Good Hope" Peters 2:76.
  • Syrmaticus reevesii (Gray,JE) 1829 Anim.Kingd.[Griffith] 8 p.25
    • Type by monotypy. Phasianus reevesii JE Gray.
    • Type locality (Peters representation): "China." Peters 2:128.
  • Somateria spectabilis (Linnaeus) 1758 Syst.Nat.ed.10 p.123
    • Type by monotypy. Anas spectabilis JE Gray.
    • Type locality: "in Canada, Svecia = Sweden" AOU CL 7ed. p.77.
  • Sasia ochracea Hodgson 1837 J.Asiat.Soc.Bengal 5(1836) p.778
    • Type by original designation.
    • Type locality: Nepal
  • Sphyrapicus varius (Linnaeus) 1766 Syst.Nat.ed.12 p.176
    • Type by original designation. Picus varius Linnaeus.
    • Type locality: "in America septontrionali = South Carolina". AOU CL 7 ed. p.337.
  • Selenidera gouldii (Natterer) 1837 PZS Pt5 no.52 p.44
    • Type by subsequent designation. S. Gouldii Natterer = Pteroglossus gouldii Natterer (Gray GR, ListGen.Birds 1840 p.5).
    • Type locality: "Para" Peters 6:80.
  • Stactolaema anchietae (Bocage) 1869 PZS p.436 pl.29
    • Type by original designation and monotypy. S. anchietae = Buccanadon anchietae Bocage.
    • Type locality (Peters representation): "Caconda, Benguella" Peters 6:43.
  • Siphonorhis americana† (Linnaeus) 1758 Syst.Nat.ed.10 p.193
    • Type by original designation and monotypy. Caprimulgus americanus Linnaeus.
    • Type locality: " in America calidiore". Ex "Sloan. jam. 2. p.298. t.255. f.1." Linnaeus.
    • Type locality (AOU 7thCL): '"Based on the Small Wood Owl" Sloane, Voy. Jamaica 2:296 pl.255. fig.1. (in America calidiore = Jamaica.)' p.269.
    • Type locality (Peters representation): '"America Calidiore" = Jamaica' Peters 4:194.
  • Semnornis ramphastinus (Jardine) 1855 Edinb.NewPhilos.J.n.s. 2 p.404
    • "New name to replace Pan Richmond, preoccupied." Peters 6:30.
    • Type locality (Peters representation): "Eastern Cordillera between Quito and Mt. Cayambe, Ecuador." Peters 6:30.
  • Streptocitta albicollis (Vieillot) 1818 Nouv.Dict.Hist.Nat. 26 p.128
    • "Type by monotypy." Peters 15:177.
    • Type locality (Peters representation): "New Caledonia, error for Buton or Muna Island." Peters 15:177 (Amadon).
    • Bonaparte gives:

    *823. Streptocitta, Bp. (Pica, p. Gr. - Garrula, p. Temm.)	Oceania.	1.
		CORVUS caledonicus! Labillardière, nec Gm. (Pica al-
		bicollis, Vieill. - Garrula torquata, Wagl.) Voy. En-
		trecast. Atl. t. 39. ex Celebes.  Nigro-cyanea: collo
		undique, pectoreque late, albis.
    • Labillardière I interpret to be
        Jacques Julien de Labillardière 1755-1834.
      A name I have not previously encounted in association with avian taxa (2002.01.01).
    • The Cat. of Books in the Brit.Mus. (N.H.) lists 5 works by M. Labillardiére. The 1800 publications of regarding the Voyage of La Péouse, or his 1824-1825 Sertum Austro-Caledonicum appear the most likely. The Sertum may well relate to plants.
    • Peters attributes Corvus caledonicus to Latham.
    • Cat.BrdsBr.Mus. 13 p.153 (Sharpe) attributes Corvus caledonicus to Latham Suppl.Ind.Orn. 1801 [=1802].
    • Therefor, it may be that Bonaparte's synonomy in this case may be correct, presuming the name was published in Labillardiere's work on the voyage of La Perous.
