Zoological Type Notes --I

  • Tinamus major (Gmelin) 1789 Syst.Nat. 1 pt2 p.767 n.63
    • Type by subsequent designation. Apstein, 1915, Sitz.Ges.Nat.FreundeBerlin no.5 p.197.
    • Type locality: Cayenne, French Guiana
  • Tinamotis pentlandii Vigors 1837 PZS(1836) p.79
    • Type by monotypy.
    • Type locality: '"on a high elevation in the Andes"; probably Bolivia.' Peters Checklist 1:47.
  • Talegalla cuvieri Lesson 1828 Man.Orn. 2 p.186
    • Type by original designation.
    • Type locality: "Forests of New Guinea = Dorey [now Manokwari] Dutch New Guinea" Peters Checklist 2:8.
  • Tetraogallus himalayensis Gray,GR 1843 PZS(1842) p.105
    • Type by monotypy.
    • Type locality: Himalayas.
    • Peters Checklist note "Tetraogallus nigellii J.E. Gray (not Lophophorus nigelli Jardine and Selby) = Tetraogallus himalayensis G.R. Gray." Peters Checklist 2:60.
  • Tetraophasis obscurus (Verreaux,J) 1869 Nouv.Arch.Mus.Hist.Nat.Bull. 5 p.33 pl.6
    • Type by original designation and monotypy Lophophorus obscurus J. Verreaux.
    • Type locality: Type from Mupin, Szechuan.
  • Tragopan satyra (Linnaeus) 1758 Syst.Nat.ed.10 p.157
    • Type by monotypy. Meleagris satyra Linnaeus.
    • Type locality: "Bengal, ex Edwards, pl.116. Error, the drawing on which Edward's plate was based was probably made from a specimen from Nepal." Peters Checklist 2:109.
  • Tetrao urogallus Linnaeus 1758 Syst.Nat.ed.10 p.159
    • Type by subsequent designation. Tetrao urogallus Linnaeus. (Gray,GR 1840 ListGen.Birds p.62).
    • Type locality: "Europe, restricted type locality, Sweden." Peters Checklist 2:24.
  • Tympanuchus cupido (Linnaeus) 1758 Syst.Nat.ed.10 p.160
    • Type by monotypy. Tetrao cupido Linnaeus.
    • "Virginia ex Catesby." Peters Checklist 2:41.
    • Type locality:"(in Virginia = Pennsylvania.)" AOU CL 7ed. p.121.
  • Indicator indicator (Sparrman) 1777 Philos.Trans. 67 p.43 pl.1
    • Type by tautonomy. Indicator Sparrmanii Stephens = Cuclus indicator Sparrman.
    • Type locality: "Great Fish River, near Somerset East, Cape Province." Peters Checklist 6:69.
  • Tachyeres brachypterus (Latham) 1790 IndexOrn. 2 p.834
    • Type by monotypy. Anas brachyptera Latham.
    • Type locality: Falkland Islands
  • Turnix sylvatica (Desfontaines) 1789 Hist.Acad.R.Sci.Paris(1787) p.500 pl.13
    • Type by subsequent designation. Tetrao gibraltericus Gmelin = Tetrao sylvaticus Desfrontaines. (Gray,GR 1840 ListGen.Birds p.63).
    • Type locality: "Near Algiers." Peters Checklist 2:142.
  • Ithaginis cruentus (Hardwicke) 1821 Trans.Linn.Soc.London(1) 13 p.237
    • Type by monotypy.
    • Type locality (Peters representation): "Mountains of Nepal". Peters Checklist 2:107.
    • The representation in Gray ListGen.Birds (1841) is here reproduced: 
      ITHAGINIS,Wagl.(1832). Plectrophorus, J.E. Gray (183?).
	Plectropus, Less.(1837). Phasianus,Hardw. Perdix,Temm.
	Tetrao,Gm. Francolinus,Deless.

  I. lunulatus, (Valenc.)G.R.Gray. P.Hardwickii,J.E. Gray.
