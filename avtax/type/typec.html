Zoological Type Notes --C

  • Calyptorhynchus banksii (Latham) 1790 IndexOrn. 1 p.107 Concept Nomenclature
    • Type by subsequent designation, C. banksii (Lath.) = Psittacus magnificus Shaw (G.R. Gray, List Gen. Birds, 1840, p.43)
    • Type locality: New Holland = New South Wales.

  • Cyanomitra verticalis (Latham) 1790 IndexOrn. 1 p.298
    • Type by subsequent designation, Certhia cyanocephala Shaw = Certhia verticalis Linnaeus (G.R. Gray Cat.Gen.Subgen.Birds 1855 p.137). [from Peters Checklist 12:223.]
    • I interpret the gender of Cyanomitra to be feminine based on Gray's use in combination with cyanocephala as the type. Reichenbach (according to the Richmond Index listing) lists the "Types as:
      • Certhia cyanocephala Shaw
        • .
      • C. pusilla Swains.
      • C. affinis Rüppell
      Therefore, I take the association with the designated type species as determinative for the gender (as the Greek adjective μτρα [mitra: banded] does not in itself indicate a gender.
    2002.09.09; 2010.11.30
    • Casuarius casuarius (Linnaeus) 1758 Syst.Nat.ed.10 p.155
      • Type by tautonomy. Casuarius = Struthio casuarius Linnaeus.
      • Type locality : "Asia, Sumatra, Molucca, Banda" (Linnaeus)
      • Type locality (Peters [Mayr] comments): 'Ceram, "corrected" type locality. Rothschild, 1900, Trans.Zool.Soc.London 15 p.115.' Peters 1:8.
    • Crypturellus tataupa (Temminck) 1815 Hist.Nat.Pig.Gallin. 3 p.590,752
      • Type by original designation. '"C. tataupa Temm." = Tinamus tataupa Temminck.'
      • Type locality (Peters [Mayr] representation): '"Bresil" = Rio de Janiero, fide Pinto, 1964, Orn.Bras. 1 p.10.' Peters 1:34.
    • Chamaepetes goudotii (Lesson) 1828 Man.Orn. 2 p.217
      • Type by monotypy, Ortalida goudotii Lesson.

    • Crax rubra Linnaeus 1758 Syst.Nat.ed.10 p.157

        • "In Ann.+Mag.N.H., XX. 1847, 319, he says rubeculoides is the type."
    • I follow the Richmond Index.
