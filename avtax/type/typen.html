Zoological Type Notes --N

  • Nystalus maculatus (Gmelin) 1788 Syst.Nat. 1 pt1 p.451
    • Type by subsequent designation, Alcedo maculata Gmelin. (Sclater, Monogr. Jacamars and Puff-birds, pt.4 1882, p.xxxv [35].
    • Type locality (Peters representation): "Brazil".
2003.02.15
  • Nystactes tamatia (Gmelin) 1788 Syst.Nat. 1 pt1 p.405
    • Type by monotypy. Bucco somnolentus Lichtenstein = Bucco tamatia Gmelin.
    • Type locality (Peters representation): "Cayenne and Brazil"
2003.02.15
  • Nothocercus julius (Bonaparte) 1854 Compt.Rend. 38 p.663
    • Type by subsequent designation. Salvadori, 1895, Cat.BirdsBrit.Mus. 27 p.509. Tinamus julius Bonaparte, 1854.
    • Type locality (Peters [Mayr] representation): "Colombia; restricted to vicinity of Bogota by Blake, 1955, Fieldiana, Zool., 37 p.15." Peters 1:18.
  • Nothoprocta perdicaria (Kittlitz) 1830 Mem.Acad.Imp.Sci.St.Petersb. 1(1831) p.192[=193] pl.12
    • Type by original designation. Sclater,PL & Salvin 1873 Nomen.Av.Neotrop. p.156 "Crypturus perdicarius Kittlitz, ex Chilia."
    • Type locality: "near Valpariso, Chile" Peters 1:38.
  • Nothura boraquira (Spix) 1825 Av.Sp.Nov.Brasil. 2 p.63 pl.79
    • Type by subsequent designation. G.R. Gray, 1840, List Gen. Birds p.63.
    • Type locality: "Type from Minas Gerais, Brazil" Peters 1:40.
  • Nettapus auritus (Boddaert) 1783 TablePlanchesEnlum. p.48
    • Type by monotypy. Anas madagascariensis Gmelin = Anas aurita Boddaert.
    • Type locality: Madagascar.
  • Netta rufina (Pallas) 1773 ReiseVersch.Provin.Russ.Reichs 2 p.713
    • Type by monotypy. Anas rufina Pallas.
    • Type locality: Caspian Sea and lakes of the Tartarian Desert.
  • Notharchus macrorhynchos (Gmelin) 1788 Syst.Nat. 1 pt1 p.406
    • Type by subsequent designation. Bucco hyperrhynchus Sclater. PZS pt.23 1856 (1855) p.193 = N. macrorhynchos hyperrhynchus (Sclater)
    • Type locality (Peters representation): "Upper Amazon". Peters 6:10.
